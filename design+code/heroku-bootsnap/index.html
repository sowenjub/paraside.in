I tried a lot of things before I finally found this reddit post, with a comment about this discussion.

And the fix was easy: use bundler v2.1.4 instead of v2.2.1.

Just check your Gemfile.lock and the version its using in the BUNDLED WITH section at the end of the file.

In my case, I simply ran gem uninstall bundler which removed v2.2.1 and kept the default v2.1.4, but here are some helpful commands for you:

gem uninstall bundler # To remove faulty versions
gem list | grep bundler # Find which bundler versions are installed on your system
gem install bundler --version '2.1.4' # Install a specific version of bundler

You also need to update your Gemfile.lock. For this I'm not sure what's the proper way to do this. I think you can simply edit the version under BUNDLED WITH. If you're just getting started, you can also trash your Gemfile.lock and run bundle install since it probably won't mess with your gems (it's what I did‚Ä¶).
